android_scripts
===============

Helpful scripts for building OptiXperia

QuickStart:

    git clone https://github.com/OptiXperia/android_scripts.git -b opti-5.1 optixperia;
    cd optixperia;
    repo init -u https://github.com/OptiPop/manifest.git -b opti-5.1;
    ./Update.sh;
